Fundamentals

What is ATE?

Every production device is tested — usually more than once. Automated Test Equipment provides deterministic stimulus and measurement so fabs and OSATs can ship quality at volume without leaking escapes to the field.

Stage 01
Wafer sort
Probe card contact; die-level binning before package
Stage 02
Final test
Packaged parts on handler; production screens
Stage 03
System-level test
Near-application exercise before ship

Why ATE exists

  • Cost of test — coverage vs. test time per unit
  • Yield — bin maps and production health
  • Escapes — catch fails before customer DOA
  • Throughput — multisite + handler cadence

Core architecture

  • Test head — pin electronics & instruments
  • DIB / loadboard — channel-to-socket routing
  • Handler / prober — mechanical contact
  • Test program — patterns, levels, timing, limits

Platforms

Types of ATE

Device class drives pin electronics, instruments, and handler strategy. Most of the market clusters into these families.

SoC / Digital

V93000 · UltraFLEX

High pin-count digital and mixed-signal SoCs.

Memory

T5500 · T5800

DRAM, NAND, and flash at wafer and final test.

Analog / Mixed-Signal

ETS / Eagle Test heritage

Precision analog, converters, and PMICs.

RF / Wireless

LitePoint · NI PXI

Cellular, Wi-Fi, and radar device test.

Power / Discrete

Chroma · Cohu Diamondx

SiC, GaN, MOSFETs, and IGBTs.

System-Level Test

SLT platforms

Near-real application context; last gate before ship.

Bench vs. Automated Test

Same physics, different constraints: exploration in the lab versus deterministic screen on the manufacturing floor.

Dimension Bench ATE
Purpose Validation, characterization, debug Production screen, yield, binning
Instrumentation Scopes, SMUs, analyzers, scripts Pin electronics, PPMU, patterns
Timing Lab / exploratory Deterministic, calibrated to DUT pin
Volume Single site, engineer-paced Multisite + handler / prober
Economics Lower CapEx, higher cost / part High CapEx, low cost / part at volume
Owner DV / characterization Product / production test

Bench validates the design. ATE screens every part that ships.
